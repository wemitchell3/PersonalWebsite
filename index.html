Projects

LugNutz

ReactJS Mobile Web App

LugNutz is designed to be a mobile web app to help users keep track of their vehicle’s maintenance and repair logs. The inspiration for this app came to me after many years of helping family and friends solve their vehicle repair problems and not having an easy way to keep track of what they and/or I had done to their vehicle over the years. LugNutz allows users to add all vehicles they are in charge of maintaining and add all previous and future maintenance/repair tasks. There is also a chat forum to allow users to share ideas and help each other diagnose and solve repair issues. LugNutz was built using React, ReactStrap | BootStrap 4, JavaScript, HTML, CSS, Firebase, & JSON.

LugNutz Premium

C# .Net Core MVC Entity Framework SQL Web App

After building LugNutz for my front-end capstone and spending the second half of my time at NSS learning the fundamentals of C#, MVC, .NET Core, Entity Framework, and SQL technologies, I determined there was much more I could add to LugNutz, so I decided to create LugNutz 2.0, premium edition.

LugNutz Premium is primarily designed to be a Desktop Web Application, geared towards garage warriors and part-time mechanics. Additional functionality includes automatic vehicle creation using the Vehicle Identification Number to fetch vehicle specifications through the National Highway Traffic Safety Administration API and vehicle specific VIN decoded reports.

Built using C#, .NET Core Entity Framework, Identity Framework, and SQL
